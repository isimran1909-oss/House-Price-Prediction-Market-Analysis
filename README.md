House Price Prediction & Market Analysis
Project Overview

This project demonstrates an end-to-end Machine Learning workflow for predicting residential property prices using Linear Regression. The project was developed to gain practical experience with data cleaning, exploratory data analysis, feature engineering, predictive modeling, and dashboard creation.

The dataset was intentionally generated in a dirty format to simulate real-world data quality issues, including missing values, inconsistent formats, duplicate records, and outliers.

Objectives
Clean and prepare raw housing data
Perform exploratory data analysis (EDA)
Identify factors influencing house prices
Build a Linear Regression model
Evaluate model performance
Visualize insights using Power BI
